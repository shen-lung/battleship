# Battleship
Battleship is the classic naval combat game that brings together competition, strategy, and excitement.

Run battleship:
1. git clone <github-repository-link>
2. yarn install
3. yarn start

Eslint:
yarn run eslint <path_to_file>

Test:
yarn jest
